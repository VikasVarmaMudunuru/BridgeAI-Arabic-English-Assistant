# BridgeAI -- Arabic ↔ English Assistant

BridgeAI is an AI-powered assistant that helps translate and interact
between **Arabic and English** through a simple web interface.

The project demonstrates a deployable AI web assistant using a
**frontend hosted on GitHub Pages** and a **backend API hosted on
Hugging Face Spaces**.

------------------------------------------------------------------------

## Deployment

### Frontend Deployment

The frontend is deployed using **GitHub Pages**.

Steps used:

1.  Push project to GitHub repository
2.  Open repository settings
3.  Navigate to **Pages**
4.  Set deployment source

Source: Deploy from branch\
Branch: main\
Folder: / (root)

GitHub then builds and publishes the site automatically.

------------------------------------------------------------------------

### Backend Deployment

The backend API is deployed on **Hugging Face Spaces**.

It handles: - AI processing - Request handling - Returning responses to
the frontend
