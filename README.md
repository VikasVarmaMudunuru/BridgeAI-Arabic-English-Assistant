# BridgeAI-Arabic-English-Assistant

BridgeAI is a simple AI-powered assistant that helps translate and interact between Arabic and English.
It provides a lightweight web interface connected to an API that processes user input and returns responses.
The project is designed to demonstrate a deployable AI web assistant using a frontend hosted on GitHub Pages and a backend API hosted on Hugging Face Spaces.

Live Application
Frontend (Web App)
https://vikasvarmamudunuru.github.io/BridgeAI-Arabic-English-Assistant/

Backend API

How the System Works
Frontend

The frontend is a static website that runs in the browser.
It sends user input to the backend API and displays the response.

Hosted on GitHub Pages.
Backend
The backend API processes the request and returns the AI-generated response.
Hosted on Hugging Face Spaces.

Flow
User Input
   ↓
Frontend (GitHub Pages)
   ↓
Backend API (HuggingFace Space)
   ↓
AI Response
   ↓
Displayed in Web Interface

How to Use the App
Open the web app:
https://vikasvarmamudunuru.github.io/BridgeAI-Arabic-English-Assistant/
Enter a message in Arabic or English.
Click Send.
The assistant processes the message and returns a response.

Running Locally
If you want to run the project locally:
1 Clone the repository
git clone https://github.com/VikasVarmaMudunuru/BridgeAI-Arabic-English-Assistant.git
2 Navigate to the folder
cd BridgeAI-Arabic-English-Assistant
3 Open the app
Open index.html in your browser.

Deployment
Frontend Deployment
The frontend is deployed using GitHub Pages.
Steps used:
Push project to GitHub repository
Open repository settings
Navigate to Pages
Set deployment source

Source: Deploy from branch
Branch: main
Folder: / (root)

Backend Deployment
The backend API is deployed on Hugging Face Spaces.

It handles:

• AI processing
• Request handling
• Returning responses to the frontend
